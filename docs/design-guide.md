## 五、关键组件详细标注与交互说明（补充设计稿细节）  

为了确保UI设计师能够精准还原交互逻辑与视觉细节，以下针对高频使用的**核心组件**（如拖放区域、滑块控件、3D画布、表格等）提供**具体尺寸、颜色、交互状态标注**，可作为设计稿中的注释或规范参考。

---

### 一、通用组件规范  

#### 1. 拖放上传区域（文件导入模块）  
- **位置**：文件导入模块的顶部中央（当未选择文件时显示）。  
- **尺寸**：宽度=父容器宽度×0.8（自适应，最小300px），高度=100px。  
- **视觉样式**：  
  - 背景：浅灰色半透明填充（RGBA: 240, 240, 240, 0.5），带虚线边框（线宽2px，虚线间隔5px）。  
  - 文字提示：居中显示，字体（系统默认无衬线，如PingFang SC/Roboto），字号14px，颜色#666666。  
  - 示例文字：“拖放CIF/TIF/ZIP文件至此处，或点击下方按钮选择文件”。  
- **交互状态**：  
  - **默认状态**：虚线边框 + 提示文字。  
  - **鼠标悬停**：边框颜色变为蓝色（#1976D2），背景透明度提高至0.8（RGBA: 240, 240, 240, 0.8）。  
  - **拖放激活**：边框颜色变为深蓝色（#0D47A1），背景填充为蓝色半透明（RGBA: 30, 136, 229, 0.1），文字提示变为“释放文件以开始上传”。  

#### 2. 滑块调节控件（图像处理模块）  
- **位置**：图像处理模块的顶部工具栏（对比度/亮度/伽马校正）。  
- **尺寸**：单个滑块宽度=120px，高度=20px（轨道） + 16px（拖动手柄）。  
- **视觉样式**：  
  - 轨道：浅灰色背景（#E0E0E0），两端为圆角矩形。  
  - 拖动手柄：蓝色圆形（#1976D2），直径16px，中心可拖动，悬停时放大至18px。  
  - 标签与数值：标签文字（如“对比度”）字号12px，颜色#333333；数值显示（如“+5%”）字号11px，颜色#666666，位于滑块右侧5px处。  
- **交互状态**：  
  - 拖动时：手柄颜色加深（#0D47A1），轨道填充当前值对应的渐变色（如对比度用蓝色渐变）。  
  - 实时反馈：画布中的图像随滑块拖动同步更新（无延迟）。  

#### 3. 3D晶体结构画布（结构可视化模块）  
- **位置**：结构可视化模块的中央内容区（点击“3D视图”后加载）。  
- **尺寸**：自适应父容器高度的70%+（最小高度400px），宽度=中央内容区宽度×0.9（居中显示）。  
- **视觉样式**：  
  - 背景：深灰色（#2D2D2D）或黑色（#000000），模拟科学可视化场景。  
  - 原子：彩色球体（根据元素类型区分颜色，如Ti-银色、O-红色），直径根据原子半径比例调整。  
  - 键长：灰色细线（#CCCCCC），连接相邻原子。  
- **交互控件（悬浮工具栏）**：  
  - 按钮组（图标+文字标签，字号11px）：  
    - 🖱️ 旋转（默认提示“拖动旋转视角”）  
    - 🔍 缩放（提示“滚轮缩放”）  
    - 🔄 重置视角（提示“双击恢复默认”）  
    - 🎯 选择原子（提示“点击查看详情”）  
  - 状态提示：底部左下角显示当前模式（如“旋转模式”） + 坐标（如“x:120, y:80”）。  

#### 4. 数据表格（CIF解析模块/成分分析模块）  
- **位置**：CIF解析模块的右侧（原子信息/晶胞参数表格）、成分分析模块的上部（元素列表）。  
- **尺寸**：宽度=父容器宽度×0.7（自适应），高度=内容区剩余空间的60%（支持滚动条）。  
- **视觉样式**：  
  - 表头：深蓝色背景（#1565C0），白色文字（#FFFFFF），字号13px，加粗。  
  - 单元格：浅灰色背景（#FAFAFA，偶数行）/ 白色（#FFFFFF，奇数行），文字字号12px，颜色#333333。  
  - 边框：单元格间细线分隔（#E0E0E0，1px）。  
- **交互功能**：  
  - 排序：点击表头字段（如“原子序数”）可升序/降序排列（表头显示▲/▼箭头）。  
  - 搜索：表格上方提供输入框（占位符“搜索元素/参数...”），输入时实时过滤行。  
  - 缺失字段：未填写的关键字段（如CIF中的空间群）背景为浅红色（#FFEBEE），文字为红色（#D32F2F），悬浮显示“⚠️ 请手动补录”。

### 二、交互状态与动效细节  

#### 1. 模块切换（左侧导航栏）  
- **交互逻辑**：用户点击左侧导航Tab（如“图像处理”→“CIF解析”），中央内容区内容在0.3秒内平滑切换（淡入淡出或滑动过渡）。  
- **视觉反馈**：当前选中的Tab高亮显示（蓝色背景#1976D2，白色文字），未选中的Tab为浅灰色文字（#666666）。  

#### 2. 文件上传成功/失败  
- **成功状态**：上传完成后，顶部状态栏显示绿色文字提示（#4CAF50）：“文件解析成功（CIF/TIF）” + 文件数量统计（如“共加载3个文件”）。  
- **失败状态**：解析失败时，状态栏显示红色文字提示（#F44336）：“CIF文件格式异常（请检查文件完整性）” 或 “TIF文件损坏”。  

#### 3. 参数调节实时预览  
- **图像处理**：拖动对比度/亮度滑块时，画布中的TIF图像实时更新（延迟＜100ms），确保用户直观看到调节效果。  
- **3D结构**：切换视图模式（线框→球棒→填充）时，模型在0.2秒内平滑过渡，避免闪烁。  

#### 4. 缺失数据提示  
- **CIF解析**：若关键字段（如空间群）未定义，表格中该单元格背景为浅红色（#FFEBEE），文字为红色（#D32F2F），悬浮鼠标时显示气泡提示：“⚠️ 此字段为必填项，请手动补录”。 点击单元格旁的“补录”按钮后，弹出输入框（居中浮层，背景白色，带确认/取消按钮）。

## 六、自适应布局关键断点与规则（量化标准）  

为确保应用在不同屏幕尺寸下均能正常使用，以下定义具体的**窗口尺寸断点**与**组件适配规则**（基于CSS媒体查询逻辑，供前端开发参考）：  

| 屏幕类型       | 窗口宽度断点 | 核心适配规则                                                                 |
|----------------|--------------|------------------------------------------------------------------------------|
| **超小屏**     | ≤1024px      | - 左侧导航栏默认收缩至50px（仅显示图标），点击图标展开临时面板（显示当前模块子功能）。<br>- 中央内容区放大至占窗口宽度的90%。<br>- 图像画布禁用非必要装饰（如阴影），表格列数减少（如合并“原子信息”子字段）。 |
| **小屏（笔记本）** | 1025px~1366px | - 左侧导航栏保持220px宽度，但按钮文字缩小至11px。<br>- 3D画布最大高度限制为窗口的60%（避免挤压表格）。<br>- 文件列表每行显示2个文件（原默认3个）。 |
| **中屏（台式机）** | 1367px~1920px | - 默认布局（左侧220px，中央自适应，顶部40px，底部20px）。<br>- 支持显示完整功能（如多维分析图表+参数面板并列）。 |
| **大屏（4K/多显示器）** | ≥1921px      | - 左侧导航栏可扩展至280px（显示更详细的模块描述）。<br>- 中央内容区允许并排显示双面板（如图像画布+右侧参数调节面板）。<br>- 表格支持横向滚动（避免压缩列宽）。 |

**通用规则**：  
- 所有文字最小字号限制为11px（避免过小影响可读性）。  
- 按钮最小点击区域为44px×44px（符合无障碍交互标准）。  
- 图像画布在任何尺寸下均保持原始宽高比（禁止拉伸变形）。
